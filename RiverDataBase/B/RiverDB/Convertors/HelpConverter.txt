База поддерживает два вида форматов данных

1.	Первый формат данных. 
Использовался в ранних наблюдениях, сторонних и моих. 
Данные хранятся в текстовом файле с расширением *.dat. 
Координаты точек определены в координатной системе 
Пулково 1942 и конвертируются при загрузке в координатную 
систему WGS84
2.	Второй формат данных.
С 2018 года используется второй формат данных с координатной 
системой WGS84, он является составным, что связано с особенностью 
вывода данных с прибора. Сначала в необходимо загрузить файл с
данными в формате *.xls которой хранит скорости, курс и т.д. но 
точность вывода координат у точек у него ниже. Для ее уточнения 
загружается файл в формате *.gpx с координатами и глубинами высокой 
точности, данные из двух источников синхронизируются по времени. 

Примечание.
Файл *.xls получаем из программы Garmin HomePort используя файл выгруженный 
из эхомапа в формате *.ADM
Файл *.gpx получаем выгружаем непосредственно из эхомапа.
