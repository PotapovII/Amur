База поддерживает два вида форматов данных

1.	Первый формат данных. 
Использовался в ранних этапах наблюдениях, сторонних и моих. 
Данные хранятся в текстовом файле с расширением *.dat. 
Координаты точек определены в координатной системе 
Пулково 1942 и конвертируются при загрузке в координатную систему WGS84
2.	Второй формат данных.
С 2018 года используется второй формат данных с координатной 
системой WGS84, он является составным, что связано с особенностью 
вывода данных с прибора. 
2.1 На первом шаге загрузить файл с данными в формате *.xls получаемый из программы HomePort,
копированием таблицы "Треки" в файл *.xls, которой хранит скорости, курс и т.д. 
Однако точность вывода координат точек наблюдения у него ниже. 
2.2. Для ее уточнения загружается файл в формате *.gpx получаемый для выбранных треков их эхомапа 
с координатами и глубинами высокой точности, данные из двух 
источников синхронизируются по времени (команда Синхронизация в БД). 
3. Очистка не синхронизированных данных выполняется после действий 1. и 2.
Сейчас выполняется автоматически, но кнопа осталась

Примечание.
Файл *.xls получаем из программы Garmin HomePort используя файл треков выгруженный 
из эхомапа в формате *.ADM
Файл *.gpx получаем выгружаем непосредственно из эхомапа.
